## Key Observations

The overall navigation was intuitive — the participant moved between pages confidently using the top navbar. The most notable friction point was the Map page filter sidebar, which the participant overlooked for a few seconds before discovering it. The lack of a confirmation dialog when disabling a course raised a concern about accidental actions, as the participant expected a "Are you sure?" prompt before the action took effect.

## Participant Feedback

- "The map is really cool, I can see how courses connect. But it's a lot to take in at first."
- "I wasn't sure if prerequisites were required when adding a course — maybe a hint or placeholder would help."
- "The Disable button is red which makes sense, but I expected it to ask me to confirm before actually doing it."
- "The Home and About pages look empty — I didn't know what the app was for until I clicked around."
