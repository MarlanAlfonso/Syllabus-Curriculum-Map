# KM Framework — Knowledge Mapping

## Framework Chosen
Knowledge Mapping

## Rationale
A curriculum is a knowledge map: it shows how knowledge units (courses) relate to 
each other, how knowledge builds from foundational to advanced, and where gaps exist. 
Making this structure visible and navigable is the core goal of this system.

## KM Concepts Applied to This Project

| KM Concept         | Definition                                       | How the App Implements It                                                  |
|--------------------|--------------------------------------------------|----------------------------------------------------------------------------|
| Knowledge Nodes    | Distinct units of knowledge in a domain          | Each course is a node with skills, outcomes, and knowledge contributions   |
| Knowledge Links    | Relationships between knowledge units            | Prerequisite edges connect courses, showing required knowledge paths       |
| Knowledge Taxonomy | Classification and categorization of knowledge   | Courses classified by yearLevel, semester, department, skillsLearned tags  |
| Knowledge Flow     | How knowledge builds over time                   | Directed graph from foundational (Year 1) to advanced (Year 4) courses     |
| Knowledge Gaps     | Missing links or isolated knowledge nodes        | Isolated nodes (no prerequisites, no dependents) are visually identifiable |
